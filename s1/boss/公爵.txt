公爵 拉撒路-h08K

说明：
本文档按当前代码和物编重新整理。鲜血男爵小游戏仍是现行机制；祭祀/牢笼类大机制已从Boss AI和机制启动入口中移除，旧版攻略中的75%固定牢笼巫妖阶段、P4鲜血宝珠吃球、触须破盾等机制，当前代码未按原流程直接对应，先不作为现行机制记录。


一、基础属性

本体：h08K，|c00883333公爵|r 拉撒路
变身：h08L，|c00883333祭祀领主|r 拉撒路

h08K属性：
1. 生命值：25000000。
2. 生命回复：50000/秒。
3. 攻击力：7530。
4. 防御：900。
5. 攻击距离：2500。
6. 攻击冷却：2秒。
7. 移动速度：500。

h08L属性：
1. 生命值：25000000。
2. 生命回复：50000/秒。
3. 攻击力：10390。
4. 防御：900。
5. 攻击距离：250。
6. 攻击冷却：1秒。
7. 移动速度：500。

召唤/关联单位：
1. h08Q 吸血鬼：6000000生命值。
2. h08P 鲜血仆人：4500000生命值。
3. h08M 鲜血男爵：4500000生命值。
4. h08O 鲜血触手怪。

特殊属性：
1. 进入战斗时设置为Boss单位。
2. 基础减伤：25%。
3. 免疫2.5秒以内的眩晕、冰冻。


二、阶段流程

1. 开场：远程形态 h08K，使用爆血术、血之藤蔓、放血、血瘟。
2. 90%：触发鲜血男爵小游戏。
3. 75%：固定变身为 h08L，切换近战技能组。
4. 祭祀/牢笼类大机制已移除，不再由Boss AI释放，也不会通过机制入口启动。
5. 25%：触发鲜血之盾。

代码中确认有90%鲜血男爵小游戏。75%时触发变身流程，用于替代原祭祀/牢笼类大机制的中段节奏点。


三、90%鲜血男爵小游戏

触发：
Boss生命低于90%时触发一次。

Boss台词：
凡人,你的xue归我了!

流程：
1. Boss进入施法引导。
2. 1.5秒后按人数召唤 h08M 鲜血男爵：1~2人召唤2只，3~4人召唤3只，5人以上召唤4只。
3. 男爵刷新位置距离Boss约1400。
4. Boss头顶显示进度：0 / 100。
5. 机制基础时间：30秒。

进度：
1. 进度变量上限为100。
2. 代码中显示为 u__1 / 100。
3. 玩家被男爵/血珠机制命中会使进度增加。
4. 男爵机制中存在两种血珠命中来源，分别增加1或2点进度。

完成/结束：
1. 进度达到100，或机制计时到30秒后进入结算。
2. 结算时Boss进入血系大爆炸读条。
3. 爆炸会影响战斗区域内敌对单位。

结算伤害：
造成魔法伤害：10000 x (1 + 0.02 x 当前进度)。

男爵单位：
1. 单位ID：h08M。
2. 名称：鲜血男爵。
3. 生命值：4500000。
4. 防御：750。
5. 攻击距离：150。
6. 移动速度：0。

男爵自身机制：
1. 鲜血男爵创建后会绑定到公爵机制。
2. 男爵周期性向Boss的进度显示物发射血珠。
3. 男爵生命低于30%或主机制结束时，会进入后续阶段。
4. 男爵后续会进入3秒血祭/死亡演出。

备注：
旧攻略里写男爵生命10000000；当前物编为4500000。


四、开场技能组：h08K

开场拥有技能：
1. A16W 公爵 - 爆血术，冷却8秒。
2. A16X 公爵 - 血之藤蔓，冷却14秒。
3. A16Y 公爵 - 放血，冷却18秒。
4. A178 公爵 - 血瘟，冷却14秒。

技能1：爆血术 A16W

定位：
点名爆炸技能。

已确认：
1. 释放时绑定A16W触发。
2. 技能基础冷却8秒。
3. 使用红色血系特效。

技能2：血之藤蔓 A16X

定位：
触手/藤蔓类召唤技能。

已确认：
1. 释放时在Boss周围生成4个方向的血之藤蔓/触须实例。
2. 方向带随机偏移。
3. 技能基础冷却14秒。
4. 释放台词：苏醒 !
5. 直线命中伤害：2500。
6. 被藤蔓缠住后每0.5秒受到 3% 最大生命值 + 500 伤害，持续约3秒。
7. 每次持续结算若命中目标，Boss恢复最大生命值的 0.5% + 每个目标0.05%。

技能3：放血 A16Y

定位：
全场吸血链接。

流程：
1. Boss施法并连接场内玩家。
2. 持续3秒。
3. 每0.5秒对被连接目标造成伤害。
4. Boss根据造成的生命损失回血。

伤害：
每次造成 5%目标最大生命值 + 500。

回血：
每次根据命中目标数量计算回血，基础为Boss最大生命百分比。

打断/中止条件：
1. Boss死亡。
2. Boss目标不可战斗。
3. Boss被限制到无法继续施法。
4. 目标死亡后对应连线移除。

技能4：血瘟 A178

定位：
点名Debuff与延迟爆炸。

流程：
1. Boss施法。
2. 给目标施加血瘟/出血类效果。
3. 持续结束后在目标位置产生爆炸。

爆炸效果：
1. 600范围。
2. 造成5000伤害。
3. 附带血系效果。
4. 若目标死亡且是玩家，可能在死亡位置继续触发一次血系冲击。


五、祭祀/牢笼类大机制（已移除）

触发：
已从Boss AI技能选择和机制启动入口中移除，现行流程不再触发该机制。

台词：
成为高尚的牺牲品..

旧机制概要：
1. Boss暂停常规行动并进入引导。
2. 场地生成环形/牢笼范围。
3. 根据玩家人数生成若干目标点或单位。
4. 玩家被拉入或分配到机制区域。
5. 机制期间会持续造成生命百分比伤害与回血互动。

人数修正：
1. 0~4名玩家：相关目标数量较少。
2. 5~8名玩家：目标数量增加。
3. 9名以上玩家：目标数量继续增加。

当前代码确认的数量规则：
1. i91oi：0~4人返回1，5~8人返回2，9人以上返回3。
2. i91Oi：0~4人返回2，5~8人返回3，9人以上返回4。
3. i91pi：机制基础时间为40秒。

处理点：
1. 尽快处理机制内生成物。
2. 避免持续站在血系区域内。
3. 机制会强化Boss回血压力。


六、75%变身

触发：
Boss生命低于75%时触发。

台词：
你的很大，力量很大 !

流程：
1. Boss开始变身演出。
2. 切换形态到 h08L。
3. 移除开场技能组：A16W、A16X、A16Y、A178。
4. 添加变身后技能组：A16Z、A170、A171、A172。
5. Boss获得额外防御/减伤类效果。

变身后获得：
1. p1oO 15。
2. p1pO 42.5。
3. 近战攻击距离250。
4. 攻击力提高到10390。


七、变身后技能组：h08L

变身后拥有技能：
1. A16Z 公爵 - 血耙子，冷却10秒。
2. A170 公爵 - 恐怖跳跃，冷却24秒。
3. A171 公爵 - 血棘，冷却14秒。
4. A172 公爵 - 血嚎，冷却40秒。

技能1：血耙子 A16Z

定位：
连续冲锋/爪击。

流程：
1. Boss锁定1500范围内目标。
2. 连续执行3次冲锋。
3. 每次冲锋前有短暂蓄力。
4. 冲锋会击退或控制命中的玩家。

伤害：
1. h08K使用时：12500。
2. 非h08K单位使用时：5000。

范围：
1. 基础宽度400。
2. 强化时范围扩大到1.5倍。

命中效果：
1. 造成伤害。
2. 眩晕1秒。
3. 附带击退。
4. 命中后有血系冲击特效。

技能2：恐怖跳跃 A170

定位：
点名跳跃与米字扩散地刺。

流程：
1. Boss选择场内玩家。
2. 目标位置出现预警圈。
3. Boss跳向目标位置。
4. 落地后造成范围伤害。
5. h08K使用时，落地后额外生成8方向扩散地刺。

跳跃次数：
1. 普通状态：1次。
2. 强化状态：2次。

重跳间隔：
1. 普通状态：2秒。
2. 强化状态：1.5秒。

伤害：
1. h08K使用时：17500。
2. 非h08K单位使用时：7500。

范围：
1. h08K使用时：400。
2. 非h08K单位使用时：300。

地刺：
1. 从落点向8个方向扩散。
2. 每条线向外延伸到约3000距离。
3. 地刺命中造成5000伤害。

技能3：血棘 A171

定位：
双点名追踪地刺。

台词：
接受命运的安排吧 !

流程：
1. Boss点名2名玩家。
2. 被点名玩家脚下生成血棘序列。
3. 先出现预警，随后生成多段地刺。
4. 最后一段为扩大范围。

伤害：
1. 普通地刺：3000。
2. 最后一段/强化地刺：7500。

范围：
1. 普通地刺：225。
2. 强化地刺：375。

处理点：
被点名玩家持续移动，不要带着地刺穿过队友。

技能4：血嚎 A172

定位：
达摩克利斯之剑类点名技能。

流程：
1. Boss开始施法。
2. 根据玩家人数选择若干目标。
3. 目标头顶出现标记。
4. 延迟后落下血剑/大剑。

普通形态：
1. 伤害：7500。
2. 范围随人数调整。
3. 目标获得短伤Debuff。

强化形态：
1. 伤害：7500 + 目标最大生命值50%。
2. 范围：600。
3. 预警时间更长。

人数与目标数：
1. 1~4人：目标数1。
2. 5~8人：目标数2。
3. 9人以上：目标数3。

范围：
1. 普通1~4人：475。
2. 普通5~7人：425。
3. 普通8人以上：375。
4. 强化：600。


八、25%鲜血之盾

触发：
Boss生命低于25%。

台词：
我在已经饥渴难耐，啊呜 !

流程：
1. Boss进入引导。
2. 生成红色护盾特效。
3. 头顶显示100%护盾。
4. 护盾值归零后进入下一段。

效果：
1. Boss获得护盾/高额减伤。
2. 护盾显示随护盾值变化。
3. 护盾结束后移除红色护盾特效。

备注：
旧版文档写“100层护盾、90%减伤、触须命中Boss减少20%护盾”。当前代码确认有100%护盾显示和高额减伤效果，但没有按旧版完整触须破盾流程直接对应。


九、AI技能选择

开场阶段：
Boss周期性恢复法力，并随机选择技能。

开场AI可选：
1. 爆血术。
2. 血之藤蔓。
3. 放血。
4. 血瘟。
5. 90%鲜血男爵小游戏通过独立血量监听触发。

血量限制：
1. 血之藤蔓在血量低于约96%后才会被AI选择。
2. 祭祀/牢笼类机制已从Boss AI中移除。
3. 血瘟在血量低于约90%后才会被AI选择。

变身后AI可选：
1. 血耙子。
2. 恐怖跳跃。
3. 血棘。
4. 血嚎。

特殊：
若Boss正在执行75%变身机制，AI会优先使用变身后技能组。
