# 掉落爆率增加规则

来源：`s1/data/war3map.j` 的掉落计算函数 `iuqoi`、击杀掉落判定段，以及 Boss 注册段。

## 掉落类型

- `OquO`：普通掉落。会参与许愿、不许愿均分、爱的放手、徽章加成等规则。
- `Oq6O`：额外/不可许愿掉落。不会被许愿筛选，不参与不许愿均分，也不吃不许愿额外倍率。

## 通用掉率加成

所有掉落先乘通用掉率倍率：

```text
基础爆率 * (1 + 掉率加成)
```

困难模式会给 Boss 配置掉率加成。例如 `h04O` 注册为 `O9pO(25.,50.)`，困难模式掉率加成是 `25%`，即乘 `1.25`。

额外/不可许愿掉落也吃这个通用掉率加成。

## 不许愿规则

对白怨火水、君主这类 `HP = 4/5` 且允许许愿的 Boss，若没有指定愿望：

```text
普通掉落实爆率 = 基础爆率 * (1 + 掉率加成) * 不许愿倍率 / 普通掉落数量
```

当前不许愿倍率为 `2.0`，即额外增加 `100%`。

普通掉落数量只统计 `OquO`，不统计 `Oq6O`。

## 指定愿望规则

对白怨火水、君主这类 `HP = 4/5` Boss：

- 有指定愿望时，其他普通掉落不会掉落。
- 指定物品不参与不许愿均分。
- 指定物品不吃不许愿额外倍率。
- 指定物品额外乘 `1.5`，即许愿额外增加 `50%`。

公式：

```text
指定物品爆率 = 基础爆率 * (1 + 掉率加成) * 爱的放手倍率 * 1.5
```

没有爱的放手时，爱的放手倍率为 `1.0`。

非 `HP = 4/5` 的旧许愿规则不同：指定物品会额外乘 `2.5`。

## 旧许愿道具说明

部分许愿道具的说明为：

```text
有愿望指定时：其他物品不会掉落，但指定物品的掉率提高150%.
无有愿望指定时：所有物品均可掉落.

灵魂宝石/结晶不受影响.
```

这对应非 `HP = 4/5` 的旧许愿规则。

### 有指定愿望

指定普通掉落时：

```text
指定物品爆率 = 基础爆率 * (1 + 掉率加成) * 2.5
```

`2.5` 表示“提高150%”，也就是变成原来的 `250%`。

其他普通掉落不会掉落。

额外/不可许愿掉落不受指定愿望影响，仍正常判定。

举例：

- 某普通掉落基础爆率 `20%`
- 普通模式，无其他掉率加成
- 指定该物品

实际爆率：

```text
20% * 2.5 = 50%
```

若困难模式提供 `25%` 掉率加成：

```text
20% * 1.25 * 2.5 = 62.5%
```

### 无指定愿望

无指定愿望时，所有普通掉落均可掉落，不会均分，也不会吃不许愿额外倍率。

公式：

```text
普通掉落实爆率 = 基础爆率 * (1 + 掉率加成)
```

举例：

- A 物品基础 `20%`
- B 物品基础 `30%`
- C 物品基础 `5%`
- 普通模式，无其他掉率加成

实际爆率仍为：

- A：20%
- B：30%
- C：5%

它们分别独立判定，可能同时掉落。

若困难模式提供 `25%` 掉率加成：

- A：25%
- B：37.5%
- C：6.25%

### 与 HP=4/5 新规则的区别

旧规则“不指定愿望”不均分：

```text
基础爆率 * (1 + 掉率加成)
```

`HP = 4/5` 新规则“不指定愿望”会均分并额外加成：

```text
基础爆率 * (1 + 掉率加成) * 不许愿倍率 / 普通掉落数量
```

当前不许愿倍率为 `2.0`。

所以同样基础 `30%` 的普通掉落：

- 旧规则无愿望：30%
- `HP = 4/5` 且 5 个普通掉落、无愿望：30% * 2.0 / 5 = 12%

但 `HP = 4/5` 指定愿望时，不吃旧规则的 `2.5`，而是：

```text
基础爆率 * (1 + 掉率加成) * 爱的放手倍率 * 1.5
```

## 爱的放手

爱的放手只影响普通掉落，不影响额外/不可许愿掉落。

- 指定给某个玩家：该玩家普通掉落倍率 `+0.8`。
- 指定给所有人：按有效人数分摊 `+0.8`。
- 最终倍率限制在 `1.0 ~ 3.0`。

爱的放手只在 `HP = 4/5` 的愿望逻辑中参与普通掉落倍率计算。

### 不放手

没有爱的放手时：

```text
爱的放手倍率 = 1.0
```

以 `h04O` 指定风灵之纹为例：

- 风灵之纹基础爆率：30%
- 普通模式掉率加成：0%
- 爱的放手倍率：1.0

实际爆率：

```text
30% * 1.0 * 1.5 = 45%
```

困难模式下：

```text
30% * 1.25 * 1.0 * 1.5 = 56.25%
```

### 单人放手给指定玩家

如果有 1 个玩家使用爱的放手，并指定给你：

```text
爱的放手倍率 = 1.0 + 0.8 = 1.8
```

`h04O` 指定风灵之纹：

普通模式：

```text
30% * 1.8 * 1.5 = 81%
```

困难模式：

```text
30% * 1.25 * 1.8 * 1.5 = 101.25%
```

### 多人放手给同一个玩家

每个指定给你的放手玩家都会提供 `+0.8`，最高封顶到 `3.0`。

`h04O` 指定风灵之纹，普通模式：

- 0 人放手：30% * 1.0 * 1.5 = 45%
- 1 人放手：30% * 1.8 * 1.5 = 81%
- 2 人放手：30% * 2.6 * 1.5 = 117%，实际判定等同必掉
- 3 人放手：30% * 3.0 * 1.5 = 135%，封顶后实际判定等同必掉

困难模式：

- 0 人放手：30% * 1.25 * 1.0 * 1.5 = 56.25%
- 1 人放手：30% * 1.25 * 1.8 * 1.5 = 101.25%，实际判定等同必掉
- 2 人放手：30% * 1.25 * 2.6 * 1.5 = 146.25%，实际判定等同必掉
- 3 人放手：30% * 1.25 * 3.0 * 1.5 = 168.75%，封顶后实际判定等同必掉

### 放手给所有人

如果放手目标是“所有人”，这 `+0.8` 会按有效人数分摊给没有指定放手对象的玩家。

例：2 个有效玩家，1 人放手给所有人。

```text
每人获得 0.8 / 2 = 0.4
爱的放手倍率 = 1.4
```

`h04O` 指定风灵之纹：

普通模式：

```text
30% * 1.4 * 1.5 = 63%
```

困难模式：

```text
30% * 1.25 * 1.4 * 1.5 = 78.75%
```

例：4 个有效玩家，1 人放手给所有人。

```text
每人获得 0.8 / 4 = 0.2
爱的放手倍率 = 1.2
```

普通模式风灵之纹：

```text
30% * 1.2 * 1.5 = 54%
```

### 放手与不许愿

爱的放手倍率也会进入 `HP = 4/5` 的不许愿普通掉落公式。

```text
普通掉落实爆率 = 基础爆率 * (1 + 掉率加成) * 爱的放手倍率 * 不许愿倍率 / 普通掉落数量
```

以 `h04O` 不许愿、普通模式、风灵之纹为例：

不放手：

```text
30% * 1.0 * 2.0 / 5 = 12%
```

1 人放手给你：

```text
30% * 1.8 * 2.0 / 5 = 21.6%
```

2 人放手给你：

```text
30% * 2.6 * 2.0 / 5 = 31.2%
```

达到封顶 `3.0`：

```text
30% * 3.0 * 2.0 / 5 = 36%
```

额外/不可许愿掉落仍不受爱的放手影响。

## 徽章加成

玩家当前徽章会对符合条件的装备/材料额外乘 `1.5`。

- `I0MD`：品质等级 `4 ~ 6` 的装备/材料。
- `I02T`：品质等级 `4 ~ 7` 的装备/材料。
- `I03X`：品质等级 `4 ~ 7` 的装备/材料，并对部分更高阶 Boss 有特殊覆盖。

该加成一般不影响白金硬币、结晶、Boss 徽章本身。

## h04O 示例

当前 `h04O 雷电战神 瓦尔托拉` 基础掉落：

- 苍穹俱宁·卸尘枷锁 `ocor`：25%，普通掉落
- 风灵之纹 `I0IG`：30%，普通掉落
- 风灵精华 `I0IH`：30%，普通掉落
- 幽光残骸 `phea`：20%，普通掉落
- 雷神徽章 `I02E`：5%，普通掉落
- 普瑞斯白金硬币 `I0O0`：24%，额外/不可许愿
- 瓦尔托拉力量结晶 `I0P0`：1%，额外/不可许愿

`h04O` 普通掉落数量为 `5`。

### 不许愿，普通模式

```text
普通掉落 = 基础爆率 * 2.0 / 5
额外掉落 = 基础爆率
```

- 苍穹俱宁·卸尘枷锁：10%
- 风灵之纹：12%
- 风灵精华：12%
- 幽光残骸：8%
- 雷神徽章：2%
- 普瑞斯白金硬币：24%
- 瓦尔托拉力量结晶：1%

### 不许愿，困难模式

困难模式再乘 `1.25`：

- 苍穹俱宁·卸尘枷锁：12.5%
- 风灵之纹：15%
- 风灵精华：15%
- 幽光残骸：10%
- 雷神徽章：2.5%
- 普瑞斯白金硬币：30%
- 瓦尔托拉力量结晶：1.25%

### 指定愿望

指定一个普通掉落时，其他普通掉落不会掉落，该指定物品不均分。

例如指定风灵之纹：

- 普通模式：45%
- 困难模式：56.25%

白金硬币、结晶不受指定愿望影响，仍按额外掉落规则判定。

### 带 I02T

`I02T 世界传奇徽章` 会让 `h04O` 的苍穹、风灵之纹、风灵精华、幽光残骸额外乘 `1.5`。

不许愿、普通模式、带 `I02T`：

- 苍穹俱宁·卸尘枷锁：15%
- 风灵之纹：18%
- 风灵精华：18%
- 幽光残骸：12%
- 普瑞斯白金硬币：24%
- 瓦尔托拉力量结晶：1%

不许愿、困难模式、带 `I02T`：

- 苍穹俱宁·卸尘枷锁：18.75%
- 风灵之纹：22.5%
- 风灵精华：22.5%
- 幽光残骸：15%
- 普瑞斯白金硬币：30%
- 瓦尔托拉力量结晶：1.25%
